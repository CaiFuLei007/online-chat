我的需求：
为我编写一个网页版本的聊天工具，类似于 微信 这种聊天软件，前端为常规 Web 页面，后端服务期望主要由 C++ 构建。

功能需求：
1. 注册和登录功能：用户可以通过邮箱进行注册，注册完成后后续可以根据设置的账号和密码直接进行登录
2. 搜索与添加功能：用户可以通过用户名称以及群聊名称搜索已经注册的其他用户，以及创建完成的群，并且可以申请添加好友，以及申请加入群聊
3. 群聊管理：用户可以自己创建群聊，创建群聊的用户是群主，群主可以同意别人申请加入群聊，也可以注销当前群聊，群成员可以发送消息，其他在群内的用户都可以看到
4. 好友聊天：用户可以通过搜索功能添加别人，成为还有的用户可以进行聊天，该聊天只有他们双方可以看到，用户可以删除自己添加的用户，后续想要再次进行聊天，需要重新进行添加
5.用户权限分类：用户分为两种普通用户和超级用户，普通用户能够看到自己已经添加的好友，以及群聊，可以与添加的好友进行聊天，也可以在加入的群聊中进行聊天；超级用户可以看到所有存在的群聊，并且可以注销任意一个群聊，对这些群聊进行管理
